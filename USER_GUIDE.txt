3. TECHNICAL & PERFORMANCE TERMS
   - Scope of work details

4. LEGAL RISK & ENFORCEMENT
   - Indemnification clauses
   - Liability limitations
   - Insurance requirements
   - Dispute resolution
   - Flow-down clauses
   - And more...

5. REGULATORY & COMPLIANCE TERMS
   - Prevailing wage requirements
   - EEO and non-discrimination

6. TROUBLESHOOTING
================================================================================

Problem: Window Flashes and Closes Immediately
-----------------------------------------------

Cause: The application encountered an error before it could display it.

Solution:
  1. Open Command Prompt (Windows Key + R, type "cmd", press Enter)
  2. Navigate to the folder: cd "C:\path\to\folder"
  3. Run: ContractAnalysisCLI.exe "path\to\contract.pdf"
  4. You'll now see the actual error message
  5. Follow the instructions in the error message


Problem: "API key not found"
-----------------------------

Cause: The config.txt file is missing or in the wrong location.

Solution:
  1. Make sure config.txt is in the SAME folder as ContractAnalysisCLI.exe
  2. Open config.txt in Notepad to verify it contains the API key
  3. The API key should be on the first line
  4. Make sure there are no extra spaces or quotes
  5. Try again


Problem: "Authentication failed" or "401 Unauthorized"
-------------------------------------------------------

Cause: The API key is invalid or expired.

Solution:
  1. Contact the person who provided the application
  2. They may need to provide a new API key
  3. They will update the config.txt file for you


Problem: "File not found"
--------------------------

Cause: The contract file path is incorrect.

Solution:
  1. Make sure the contract file exists
  2. Try using the full path: C:\Users\YourName\Documents\contract.pdf
  3. If the path has spaces, use quotes: "C:\My Documents\contract.pdf"
  4. Try copying the contract to the same folder as the .exe


Problem: Analysis Takes Too Long
---------------------------------

Expected Times:
  - Text-based PDF (10 pages): ~1 minute
  - Scanned PDF (15 pages): 3-4 minutes
  - Word document (10 pages): ~1 minute

If it takes longer:
  1. Check your internet connection
  2. Large contracts (50+ pages) may take longer
  3. Scanned/image PDFs take much longer (requires OCR)
  4. Wait at least 5 minutes before canceling


Problem: "Tesseract not found" (for scanned PDFs)
--------------------------------------------------

Cause: You're trying to analyze a scanned/image-based PDF without OCR software.

Solution:
  1. Most contracts don't need this - try a different contract first
  2. If you need to analyze scanned PDFs, contact your IT department
  3. They can install Tesseract OCR software
  4. Or convert the scanned PDF to text first using Adobe Acrobat


Problem: No Output File Created
--------------------------------

Cause: Analysis failed or was interrupted.

Solution:
  1. Check the console window for error messages
  2. Make sure you pressed Enter to close (don't close the window manually)
  3. Check if you have write permissions in the contract's folder
  4. Try saving the contract to a different location (e.g., Desktop)
  5. Run the analysis again


Problem: Can't Open JSON File
------------------------------

Cause: No default program is set for .json files.

Solution:
  1. Right-click on the .json file
  2. Select "Open with"
  3. Choose "Notepad" or "Notepad++"
  4. Check "Always use this app to open .json files"
  5. Click OK

8. FREQUENTLY ASKED QUESTIONS
================================================================================

Q: How much does it cost to analyze a contract?
A: The cost depends on the contract size. Typically:
   - 10-page contract: ~$0.01-0.02
   - 20-page contract: ~$0.02-0.03
   - 50-page contract: ~$0.03-0.05
   The person who provided the application covers these costs.

Q: Can I analyze the same contract multiple times?
A: Yes! Each analysis will overwrite the previous results file.

Q: What file formats are supported?
A: - PDF files (.pdf) - both text-based and scanned
   - Word documents (.docx)
   - NOT supported: .doc (old Word), .txt, images

Q: Is my contract data secure?
A: The contract is sent to OpenAI's servers for analysis. OpenAI has
   security measures in place, but you should only analyze contracts
   you're authorized to share with a third-party service.

Q: Can I use this offline?
A: No, an internet connection is required for the AI analysis.

Q: Can I analyze contracts in other languages?
A: The tool is optimized for English contracts. Other languages may
   not produce accurate results.

Q: What if the analysis is wrong?
A: The AI provides analysis based on the contract text, but it's not
   a substitute for legal review. Always have important contracts
   reviewed by a qualified attorney.

Q: Can I customize the analysis?
A: No, the analysis follows a predefined structure. Contact the person
   who provided the application if you need different analysis criteria.

Q: How do I update the application?
A: You'll receive a new ContractAnalysisCLI.exe file when updates are
   available. Replace the old file with the new one. Keep the same
   config.txt file.

Q: Can I install this on multiple computers?
A: Yes, just copy both files (ContractAnalysisCLI.exe and config.txt)
   to each computer. They must stay together in the same folder.

Q: What happens if I lose the config.txt file?
A: Contact the person who provided the application. They can give you
   a new config.txt file.
