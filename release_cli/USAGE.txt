Contract Analysis Tool - CLI Version
====================================

USAGE:
------

Method 1: Command Line
  ContractAnalysisCLI.exe "path\to\contract.pdf"

Method 2: Drag and Drop
  Drag a PDF or DOCX file onto ContractAnalysisCLI.exe

REQUIREMENTS:
-------------
1. OpenAI API Key (required)
   - Set environment variable: OPENAI_API_KEY
   - See API_KEY_SETUP.txt for instructions

2. Tesseract OCR (optional, for scanned PDFs)
   - See OCR_SETUP_GUIDE.txt for installation

3. Poppler (optional, for scanned PDFs)
   - See OCR_SETUP_GUIDE.txt for installation

OUTPUT:
-------
- [filename]_analysis.json  (structured data)
- [filename]_analysis.pdf   (professional report)

Both files are saved in the same folder as the input file.

EXAMPLES:
---------
  ContractAnalysisCLI.exe "C:\Documents\Contract.pdf"
  ContractAnalysisCLI.exe "Service Agreement.docx"

TROUBLESHOOTING:
----------------
- If API key error: Set OPENAI_API_KEY environment variable
- If OCR fails: Install Tesseract and Poppler
- Check error.log for detailed error messages

For detailed setup instructions, see:
- API_KEY_SETUP.txt
- OCR_SETUP_GUIDE.txt
- README.txt
